# Atlas Exchange Demo Video

Duration target: 75 seconds.

## Scene List

1. **Cold open, 0-8s**
   - Message: "London exchange, planned before departure."
   - Visual: cinematic London background, Atlas Exchange mark, slow push.

2. **Problem, 8-18s**
   - Message: "Housing, budget, packing, deadlines. Usually scattered."
   - Visual: four cards appear one at a time, not all at once.

3. **Destination selection, 18-30s**
   - Message: "Choose London. Pick UCL."
   - Visual: region globe, partner university cards, UCL selected.

4. **Student intake, 30-42s**
   - Message: "Budget, stay length, housing preference."
   - Visual: form values animate into a clean profile card.

5. **Agent result, 42-60s**
   - Message: "Atlas returns a living plan."
   - Visual: accommodation ranking, budget estimate, packing checklist, deadline strip.

6. **Trust layer, 60-70s**
   - Message: "Every recommendation carries a source."
   - Visual: source cards with platform links and fetched timestamps.

7. **Close, 70-75s**
   - Message: "London first. Global exchange paths next."
   - Visual: regions fade in around Atlas Exchange.

## Video Rules

- Keep text inside 80px side safe area and 100px top/bottom safe area.
- Use one main message per scene.
- Do not shrink text to fit more widgets.
- Use Remotion `interpolate()` and inline style transforms.
- Do not use CSS transitions or CSS animations in Remotion.
- Use the same dark Atlas Exchange palette as the app.
